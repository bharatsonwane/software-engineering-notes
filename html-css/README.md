# HTML & CSS Notes

Notes for building web pages with HTML and CSS.

Each subfolder uses **numbered files in learning order**:

```
01-html-document-structure.md
01-css-syntax-selectors.md
...
```

Topic prefix varies per folder (`html`, `css`). See each folder's `README.md` for the full file list.

**Suggested order:** complete **HTML** (files 01–08) before deep **CSS layout** (Flexbox/Grid). JavaScript notes assume basic HTML/CSS (see [JavaScript](../javascript/)).
